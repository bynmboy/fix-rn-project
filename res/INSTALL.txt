
	Este é um instalador para o programa "fix-rn-project" para fim de facilitar
	o uso do mesmo.

	Este instalador é facilmente legível e decifrável para todo e qualquer nível
	de usuário/escritor/programador Shell, ele fará uma verificação da existência
	dos respectivos diretórios para a criação e posterior instalação.

	A instalação basicamente irá copiar os arquivos do diretório baixado para os
	diretórios criados com base na necessidade de organização. Por padrão meus
	programas em serão armazenados no diretório "~/.fc-software" para melhor
	controle e organização dos mesmos.

	Resumo de uso:

		fix-rn-project argumentos

			-h ou --help

				Irá mostrar este documento com a finalidade de orientar e 
				auxiliar os usuários no manuseio do programa.

			-i ou --install

				Irá fazer o processo de instalação do software de forma
				organizada e estruturada sem poluição do sistema.

			-v ou --version

				Mostra a versão atual do Software.

			-a ou --autor

				Mostra o Autor do Software.

			-b ou --bornin

				Mostra a data de criação do Software.

			-d ou --dir

				Mostra onde será feita a instalação do Software.
